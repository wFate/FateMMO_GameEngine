# Chat UI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add an in-game chat panel that displays system messages, trade notifications, gauntlet announcements, and player chat — making all server-side features visible to the player.

**Architecture:** A new `ChatUI` class in `game/ui/chat_ui.h/.cpp` renders an ImGui window with a scrolling message log and text input. Messages arrive via the existing `NetClient::onChatMessage` callback. The chat panel is toggled with Enter key (already mapped to `ActionId::Chat`). Channel tabs filter by message type. Uses `GetForegroundDrawList` for any overlay rendering (per project feedback: background draw list is invisible behind editor viewport).

**Tech Stack:** C++20, Dear ImGui, existing NetClient callback system, existing ActionMap input

### Task 1: ChatUI Message Buffer and Data Types

**Files:**
- Create: `game/ui/chat_ui.h`

- [ ] **Step 1: Create the header with message struct and class**

```cpp
#pragma once
#include <string>
#include <vector>
#include <cstdint>
#include "imgui.h"

namespace fate {

struct ChatMessage {
    uint8_t channel = 0;     // ChatChannel enum value
    std::string senderName;
    std::string message;
    uint8_t faction = 0;
    ImVec4 color = {1, 1, 1, 1};
};

class ChatUI {
public:
    static constexpr int MAX_MESSAGES = 200;
    static constexpr int MAX_INPUT_LENGTH = 200;

    void addMessage(uint8_t channel, const std::string& sender,
                    const std::string& message, uint8_t faction);
    void render(float displayWidth, float displayHeight);
    void toggle() { visible_ = !visible_; }
    void setVisible(bool v) { visible_ = v; }
    bool isVisible() const { return visible_; }
    bool wantsInput() const { return visible_ && inputFocused_; }

    // Set callback for when player sends a message
    std::function<void(uint8_t channel, const std::string& message)> onSendMessage;

private:
    std::vector<ChatMessage> messages_;
    char inputBuffer_[MAX_INPUT_LENGTH + 1] = {};
    bool visible_ = false;
    bool inputFocused_ = false;
    bool scrollToBottom_ = false;
    int activeTab_ = 0; // 0=All, 1=Map, 2=Global, 3=Trade, 4=Party, 5=Guild, 6=System

    ImVec4 getChannelColor(uint8_t channel) const;
    std::string getChannelPrefix(uint8_t channel) const;
    bool shouldShowMessage(uint8_t channel) const;
};

} // namespace fate
```

```cpp
chatUI_.onSendMessage = [this](uint8_t channel, const std::string& message) {
    // Build CmdChat and send
    CmdChat cmd;
    cmd.channel = channel;
    cmd.message = message;
    uint8_t buf[512];
    ByteWriter w(buf, sizeof(buf));
    cmd.write(w);
    netClient_.send(Channel::ReliableOrdered, PacketType::CmdChat, buf, w.size());
};
```
